# 你的输出格式不正确

## 问题分析
你的上次输出：`{{ last_output }}`

既不是有效的工具调用 JSON，也不是控制信号 JSON。

## 必须的输出格式
你现在**必须**且**只能**输出以下两种格式之一：

### 1. 如果步骤已完成：
```json
{"control":"step_done"}
```

### 2. 如果无法继续需要重规划：
```json
{"control":"replan","reason":"具体原因说明"}
```

## 严禁事项
- ❌ 禁止输出任何解释、开场白、总结文字
- ❌ 禁止输出自然语言描述
- ❌ 禁止输出多余的工具调用
- ❌ 禁止输出代码块包裹

## 强制要求
立即且仅输出控制信号 JSON，不要任何其他内容。
